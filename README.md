# Ibrahim Malik

**PhD Candidate** · Trinity College Dublin &nbsp;|&nbsp; **Pre-Doctoral Researcher** · IBM Research Ireland

I work at the intersection of AI and software security, with a focus on the security of LLM-generated code, static analysis, abstract interpretation, and LLM-based vulnerability detection. I contribute to the [SecQDevOps](https://secqdevops.eu/) Horizon Europe project on security in quantum software development pipelines, supervised by [Dr. Giulio Zizzo](https://research.ibm.com/people/giulio-zizzo) (IBM Research) and [Prof. John Kelleher](https://www.adaptcentre.ie/experts/john-d-kelleher/) (TCD).

**Research interests:** LLM-generated code security &nbsp;·&nbsp; SAST evaluation &nbsp;·&nbsp; Abstract interpretation &nbsp;·&nbsp; Vulnerability detection &nbsp;·&nbsp; Quantum software security
